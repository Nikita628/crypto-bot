// to start the project
docker-compose -f docker-compose-local.yml up --build -d


1. install docker
2. run the above command to start containers
3. open pgAdmin, add new server with settings (settings for pgAdmin are in docker-compose file)
4. apply migration script to the DB
5. test, make sure all is running (you can make test requests to api by opening your browser localhost:5010/create-history-data)
